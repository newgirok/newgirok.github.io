LoadingScene вҖ” м—җм…Ӣ лЎңл”© + м• лӢҲл©”мқҙм…ҳ л“ұлЎқ

мқҙм „ нҸ¬мҠӨнҠём—җм„ң лӢӨлӨҳлӢӨ. лЎңл”© лҒқлӮҳл©ҙ л°”лЎң mainSceneмңјлЎң м „нҷҳ.

create() {
  this.scene.start("mainScene");
  // м• лӢҲл©”мқҙм…ҳ л“ұлЎқ (м „м—ӯ кіөмң )
  this.anims.create({ key: "player_anim", ... });
  this.anims.create({ key: "mob1_anim", ... });
  // ...

animsлҠ” м „м—ӯмңјлЎң л“ұлЎқлҗҳкё° л•Ңл¬ём—җ LoadingSceneм—җм„ң н•ң лІҲл§Ң л§Ңл“Өл©ҙ лӢӨлҘё м”¬м—җм„ңлҸ„ к·ёлғҘ м“ё мҲҳ мһҲлӢӨ.

PlayingScene вҖ” кІҢмһ„мқҳ м „л¶Җ

мӢӨм ң кІҢмһ„мқҙ лҸҢм•„к°ҖлҠ” м”¬мқҙлӢӨ. create()м—җм„ң мҙҲкё°нҷ”, update()м—җм„ң л§Ө н”„л Ҳмһ„ к°ұмӢ .

create() вҖ” н•ң лІҲл§Ң мӢӨн–үлҗҳлҠ” мҙҲкё°нҷ”

create() {
  // мӮ¬мҡҙл“ң л“ұлЎқ
  this.m_beamSound = this.sound.add("audio_beam");
  this.m_scratchSound = this.sound.add("audio_scratch");
  // ...

  // н”Ңл Ҳмқҙм–ҙ мғқм„ұ
  this.m_player = new Player(this);

  // л°°кІҪ
  setBackground(this, "background1");

  // нӮӨліҙл“ң мһ…л Ҙ
  this.m_cursorKeys = this.input.keyboard.createCursorKeys();

  // м№ҙл©”лқјк°Җ н”Ңл Ҳмқҙм–ҙлҘј л”°лқјлӢӨлӢҳ
  this.cameras.main.startFollow(this.m_player);

  // лӘ¬мҠӨн„° к·ёлЈ№
  this.m_mobs = this.physics.add.group();

  // л¬ҙкё° к·ёлЈ№ (dynamic = лӮ м•„к°ҖлҠ” кІғ, static = кі м • мң„м№ҳ)
  this.m_weaponDynamic = this.add.group();
  this.m_weaponStatic = this.add.group();

  // кІҪн—ҳм№ҳ мҳӨлёҢм қнҠё к·ёлЈ№
  this.m_expUps = this.physics.add.group();

  // м¶©лҸҢ л“ұлЎқ
  this.physics.add.overlap(this.m_player, this.m_mobs, ...);
  this.physics.add.overlap(this.m_weaponDynamic, this.m_mobs, ...);
  this.physics.add.overlap(this.m_weaponStatic, this.m_mobs, ...);
  this.physics.add.overlap(this.m_player, this.m_expUps, ...);

  // UI
  this.m_topBar = new TopBar(this);
  this.m_expBar = new ExpBar(this, 50);
}

м¶©лҸҢмқ„ physics.add.overlapмңјлЎң л“ұлЎқн•ҳлҠ” кІҢ Phaserмқҳ л°©мӢқмқҙлӢӨ. м¶©лҸҢмқҙ л°ңмғқн•ҳл©ҙ мҪңл°ұмқҙ нҳём¶ңлҗңлӢӨ. м—¬кё°м„ң лҚ°лҜём§Җ мІҳлҰ¬, кІҪн—ҳм№ҳ нҡҚл“қ лЎңм§Ғмқҙ мӢӨн–үлҗңлӢӨ.

update() вҖ” л§Ө н”„л Ҳмһ„ к°ұмӢ 

update() {
  this.movePlayerManager();

  // л°°кІҪ нғҖмқјмқ„ н”Ңл Ҳмқҙм–ҙ мң„м№ҳм—җ л§һкІҢ мҠӨнҒ¬лЎӨ
  this.m_background.tilePositionX = this.m_player.x - Config.width / 2;
  this.m_background.tilePositionY = this.m_player.y - Config.height / 2;

  // к°ҖмһҘ к°Җк№Ңмҡҙ лӘ¬мҠӨн„°лҘј м°ҫм•„м„ң л¬ҙкё°к°Җ мЎ°мӨҖн•  мҲҳ мһҲкІҢ м ҖмһҘ

update()лҠ” мөңлҢҖн•ң к°ҖліҚкІҢ мң м§Җн–ҲлӢӨ. мӢӨм ң лЎңм§ҒмқҖ л§ӨлӢҲм Җ н•ЁмҲҳл“Өм—җ мң„мһ„н•ңлӢӨ.

л ҲлІЁм—… мӢңмҠӨн…ң вҖ” afterLevelUp()

кІҪн—ҳм№ҳк°Җ кҪү м°Ёл©ҙ pause(this, "levelup")мңјлЎң мқјмӢңм •м§Җ м”¬мқҙ лңЁкі , м„ нғқ нӣ„ afterLevelUp()мқҙ нҳём¶ңлҗңлӢӨ.

afterLevelUp() {
  this.m_topBar.gainLevel();

  switch (this.m_topBar.m_level) {
    case 2:
      removeOldestMobEvent(this);
      addMobEvent(this, 1000, "mob2", "mob2_anim", 20, 0.8);
      setAttackScale(this, "claw", 4);
      break;
    case 3:
      addAttackEvent(this, "catnip", 10, 2);
      break;
    // ...
    case 7:
      addMob(this, "lion", "lion_anim", 200, 0); // ліҙмҠӨ л“ұмһҘ
      setBackground(this, "background2");
      break;
  }
}
