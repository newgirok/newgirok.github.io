`fzf`에 `--preview` 옵션을 붙이면 git 커밋을 탐색하면서 동시에 diff를 볼 수 있다.

```bash
git log --oneline | fzf --preview 'git show --stat --color=always {1}'
```

`{1}`은 fzf가 선택한 라인의 첫 번째 필드(커밋 해시)를 의미한다.

더 실용적인 버전 — diff 전체 내용 보기:

```bash
git log --oneline | fzf --preview 'git show --color=always {1}' --preview-window=right:60%
```

### 왜 유용한가

`git log --oneline` 후 커밋 해시를 복사해서 `git show`를 따로 치는 과정이 없어진다.  
특히 오래된 커밋을 찾아볼 때 체감 속도가 다르다.
